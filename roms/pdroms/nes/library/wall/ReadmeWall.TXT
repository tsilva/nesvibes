This is an NES demo that I made to show off some of the NES' features.
This particular demo displays a wavy effect coupled with the "Colour
Emphasis" bits of the NES to simulate translucency.  So far, no emulator
can display this demo completely right, although it works fine on a real
NES.  In time, maybe the emphasis bits will be emulated correctly.

I've included the asm source for you to look at.  You should assemble it
with the cross-assembler DASM for good results.  To make it, assemble the
source, and join the header to the output file at the front, and the CHR
file at the end.
